# ИТМО: История корпусов
## Структура проекта

```
itmo-history/
├── index.html              # Главная страница
├── buildings/              # Страницы отдельных корпусов
│   ├── kronverksky.html    # Кронверкский пр., 49
│   ├── lomonosova.html     # ул. Ломоносова, 9
│   ├── birzhevaya.html     # Биржевая линия, 14
│   ├── grivtsova.html      # пер. Гривцова, 14
│   └── tchaikovsky.html    # ул. Чайковского, 11
├── css/
│   └── style.css           # Все стили
├── js/
│   ├── main.js             # Главная логика: карта, карточки, таймлайн
│   └── building.js         # Логика страниц корпусов
├── data/
│   └── buildings.json      # Данные о корпусах (история, координаты, факты)
└── images/                 # Изображения (добавить вручную)
```
